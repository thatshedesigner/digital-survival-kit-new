## 📌 Project Overview
This project is a Digital Literacy Portfolio created as part of the Digital Ambassador role. It aims to help students understand essential digital skills such as online presence, communication, collaboration, and cybersecurity.

The project is structured into five tasks, each focusing on a key aspect of digital literacy.

---
