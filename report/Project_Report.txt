Introduction:
This project focuses on digital literacy and its importance for students. It aims to help students understand how to use digital tools safely and effectively.

Task 1:
I created an infographic using Canva explaining digital literacy, tools, safety, and email etiquette. It helped me understand how to present information visually.

Task 2:
I created profiles on GitHub and LinkedIn. These platforms help in building a professional online presence.

Task 3:
I solved a coding problem on HackerRank and created a Google Form quiz. These tools are useful for learning and collaboration.

Task 4:
I wrote professional emails and created a social media checklist. This helped me understand proper online communication.

Task 5:
I studied UPI fraud and created a prevention checklist. It helped me learn how to stay safe online.

Conclusion:
This project improved my understanding of digital tools, communication, and online safety.
